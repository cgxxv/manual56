### 4.6.13 mysql\_fix\_extensions — 规范化表文件扩展名

[mysql\_fix\_extensions](#) 将MyISAM（或者ISAM)表文件扩张名转换成标准形式，它查找扩展名大小写为.frm .myd .myi .isd 和 .ism的文件然后相应的重命名成扩张名为 .frm .MYD .MYI .ISD 和 .ISM。当将文件从大小写不敏感的系统（如Windows）转移到大小写敏感的系统时该功能将非常有用。

[mysql\_fix\_extensions](#) 的调用如下，*data_dir* 是MySQL数据所在目录的路径名：

```shell
shell> mysql_fix_extensions data_dir
```
